char_heightHeight of characters in the font, in subpixels.
filenameName of the .pbm file from which the font is loaded. Each character in the file should be drawn as char_width x char_height pixels with white pixels as foreground and black pixels as background. The image should contain 16 rows of 16 characters (256 total) in ASCII order.
